Тема 7.

Графічний інтерфейс

Теорія Завдання

Tkinter

Багато програм на сьогоднішній день використовують графічний інтерфейс,
який більш інтуїтивний та зручний для користувача, ніж консоль.
І за допомогою мови програмування Python можна створювати графічні програми.
Для цього в Python за замовчуванням застосовується спеціальний набір компонентів,
який називається tkinter.

Для того щоб отримати звичайне вікно потрібно прописати:

В підсумку отримаємо

Початкова позиція вікна

За замовчуванням вікно позиціонується у відповідному місці на екрані.
Але ми можемо змінити його положення, передавши потрібні значення метод geometry().
Тепер рядок у методі geometry() має такий формат:
"Ширина x Висота + координата X + координата Y"

Кнопка Button()

Для того щоб добавити конопку потрібно прописати:

Візуалізація кнопки

Обробник подій

Для обробки натискання на кнопку необхідно встановити в конструкторі параметр command,
надавши йому посилання на функцію, яка спрацьовуватиме при натисканні.
Важливо: В параметрі command може оброблятись тільки функція!
